# Editor tutorial
## Keybinds
- C to copy scene as json string to clipboard (requires confirmation)
- V to paste scene json string (requires confirmation)
- X to clear scene (requires confirmation)
- E to toggle collision rendering
- R set individual collisionRule
- E to toggle collision rendering
- Up and down arrow keys to change collisionRule (only works if collision rendering is on)

## Selection keybinds
- F, D, H , T to start selection
- F again to delete selected area
- G again to place selected tile in selected area
- H prompts for a number 0-1 (chance for each tile) to fill selected area with
- T set collisonRule for area to selected collisionRule

## Camera Keybinds
- WASD to move camera

## Mouse controls
- Space to choose Sprite to build
- Left mouse button: place sprite
- Right mouse button: remove object
- Left mouse button + Left shift: place sprite on top of current sprites
- Right mouse button + Left shift: remove top sprite
